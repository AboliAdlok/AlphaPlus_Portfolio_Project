AlphaPulse is a financial analytics project that helps asset managers monitor and manage portfolio risk in real time by analyzing stock performance, volatility, and asset correlations. 
It calculates key metrics like daily returns, rolling volatility, and Value at Risk (VaR), while using Monte Carlo simulations (10,000+ runs) to forecast the range of possible future portfolio values. 
The project also includes interactive what-if analysis to simulate market shocks (such as a sector decline) and visualize their impact on the overall portfolio. 
Built using Python (for data collection and statistical modeling with NumPy) and a visualization tool like Tableau/Power BI, the dashboard provides clear, data-driven insights for better investment decision-making and risk management.
